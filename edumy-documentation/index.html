Edumy - Education App, Quiz, Badgewith Admin Panel - Create and sell your course using Edumy App

Edumy Education App is app that built using flutter with power full function,clean code and have beautiful design like udemy, this app have 3 rule important, the first rule is admin, teacher/author and user. Admin can add unlimited teacher/author via admin panel, approve course that uploaded by teacher/author. Teacher/author can upload unlimited course via admin panel with their account. User can buy course directly in Edumy app using payment gateway braintree(paypal, credit card, google pay and more).

Requirements

There are system requirements to setup this app to working perfectly, please follow step by step.

Recomended PHP Configuration Limits:

  • upload_max_filesize = 32M (change to higher when upload higher size of video)
  • post_max_size = 32M (change to higher when upload higher size of video)
  • PHP Version: 8.0
  • Flutter 3.13.5 • channel stable and Dart 2.19.2

Braintree Account Setup:

  • Please register first in braintree
  • Setup you payment gateway on braintree dashboard
  • Copy and paste your api Merchant Id, Public Key and Private Key

If you dont know how to change that, please contact your hosting provider to help you.

 

Database

Firstly create new database on your phpmyadmin, after that open db directory and import it.

if you have imported database to your phpmyadmin, please change database connection in include/connection.php

And the last change base url of image, the location file of base url img in image/baseurl.php

 

Admin Panel

Admin panel have 2 rule, admin and teacher/author to create course or manage course easily, before create course and publish it make sure change the braintree integration in setting button and change with yours.

Go ahead to manage course, you can click on course button on top button or click course on home page.

 

Edit Button (green button) used to edit course, edit status approved or rejected, change title, discount, price, image and more

Add Button (blue button) used to add new lecture on course

Trash Button (red button) used to delete course (when you delete this course, all lecture will be deleted to)

If you dont understand how to deploy to hosting, you can follow video tutorial on youtube by access this link Deploy to hosting

To old buyer please follow these step to modify your code

1. Import multiselect-dropdown.js in (assets/js/multiselect-dropdown.js) to your current website
2. Update header in (inlcude/header.php) please take look in line 36
3. You can delete vendor directory and composer, because in latest project no need that again (braintree)
4. Replace transaction file in (uitls/transaction.php) with latest version
5. Update home.php with latest version (copy and paste to your website)
6. Copy and paste addenrollment.php and (action/add_enrollment.php) to your website
7. Finish!!!
